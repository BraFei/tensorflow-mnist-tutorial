# cs496week3_mnist
MNIST classification in Tensorflow using Django

## Requirements
- Python 3
- Tensorflow > 0.12
- Numpy
- Django (2, 0, 1, 'final', 0)


## Training
```bash
python train.py
```


## Run Server
```bash
python train.python manage.py runserver 0.0.0.0:8000
```
